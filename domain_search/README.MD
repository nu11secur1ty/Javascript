## Usage of `domain_search` for int.domain.int> forward and backward

1. Install node
- For Kali Linux
```bash
apt install nodejs
```
3. Download the app
```bash
wget https://raw.githubusercontent.com/nu11secur1ty/Javascript/master/domain_search/izrud.sh
```
4. Run izrud.sh
5. check result.txt

## Done =)
